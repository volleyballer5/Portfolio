Core4 Definitions

Conditionals

A conditional carries out a certain action based on whether or not a given statement is true or false.
ex. if x == 1:

A real life analogy could be if the stoplight is green then proceed.

Functions

A function can be defined to perform a specific task when it is inserted.
ex. def func1()

A real life analogy could be a morning routine that has certain set tasks that someone perfoms every morning.

Variables

A variable is a value that can be set and changed during a program.
ex. x y

A real life analogy could be that there is four pieces of bread. The variable
bread would equal four but for every slice eaten, the variable(bread) decreases by 1.

Loops

A loop run a piece of code for a certain number of times or until a condition is met.
ex. while x == 1

While the stoplight green proceed throught the light.

Other Terms

Lists

A list is a compilation data in a certain order. Lists are useful because you can run through
the data in a list without indivually writing out a code to run through each item.

Tuples

A tuple is a sequence of immutable objects which means that tuples can not be changed, unlike lists.
Tuples use parentheses as opposed to the brackets that lists use.
Tupels group related data and are most commonly used to store the rgb info of pixels in pictures.
